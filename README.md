# Baytak App — React Native Expo

تطبيق بيتك للخدمات المنزلية — مبني بـ React Native Expo

## الشاشات المكتملة
- Splash Screen
- Onboarding (3 شاشات)
- Login
- Choose Account Type
- Create Account
- Verification (OTP)

## كيفية تشغيل التطبيق

```bash
cd baytak-app
npm install
npx expo start
```

ثم:
- اضغط `w` للتشغيل على المتصفح (Web)
- اضغط `i` لـ iOS Simulator
- اضغط `a` لـ Android Emulator
- افتح تطبيق **Expo Go** على هاتفك وامسح QR Code
